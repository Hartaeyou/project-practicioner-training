# FinTrust AI

Trust-Based Credit Scoring untuk memperluas akses pembiayaan UMKM.
Tim P0248 — Hackathon Bank Indonesia.

## Struktur Proyek
- `apps/mobile-umkm` — aplikasi mobile untuk UMKM (React + Vite)
- `apps/web-lender` — dashboard web untuk lender (React + Vite)
- `apps/ai-service` — engine scoring & API (Python)

## Cara Menjalankan
Masing-masing app dijalankan terpisah, lihat README di tiap folder.
